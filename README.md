# Shane Barry's Vim Configuration

Here is the vim configuration that I use. This is a work in progress and will be tweaked and changed.

## Install locally

Clone the repo

```bash
$ git clone https://github.com/Shane325/vim.git ~/.vim
```

Create symlink

```bash
$ ln -s ~/.vim/.vimrc ~/.vimrc
```
